# 概要

- Vue3, TypeScript, Vite を用いたWebアプリケーションです。
- ドキュメントの作成・編集・保存・エクスポート（HTML/PDF/JSON/TEXT）機能を備えています。
- 創作活動を支援するツールとして、キャラクター名の変換機能などを設けています。UI/UXに配慮し、直感的な操作が可能です。

## 主な機能

- ドキュメントの新規作成・編集・削除
- テキスト装飾や色付けなどのリッチエディタ機能
- 作成したドキュメントのHTML/PDF/JSON/TEXT形式でのエクスポート
- 最近使った色を保存・再選択できる機能
- シンプルで見やすいサイドバー・ツールバー

## 技術スタック

- フロントエンド: Vue3, TypeScript, Vite
- スタイル: CSS
